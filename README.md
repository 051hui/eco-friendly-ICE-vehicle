## 📌 프로젝트 개요
친환경성과 효율성을 동시에 만족하는 내연기관 차량의 조건을 데이터 분석을 통해 도출한 프로젝트입니다.
차량의 연비 및 CO₂ 배출량 데이터를 기반으로, 주요 설계 요소(엔진, 실린더, 연료 등)가 성능에 미치는 영향을 분석했습니다.

## ❗ 문제 정의
전기차 화재, 충전 인프라 부족 등의 문제로 인해 기존 내연기관 차량의 친환경성 개선 필요성이 대두되었습니다.
이에 따라, **“가장 친환경적이면서 효율적인 내연기관 차량의 조건은 무엇인가?”** 를 핵심 문제로 설정했습니다.

## 📊 데이터
- 2000~2022년 캐나다 경량 승용차 데이터
 - 주요 변수
 - ENGINE SIZE (엔진 크기)
 - CYLINDERS (실린더 수)
 - FUEL TYPE (연료 종류)
 - COMB (연비)
 - CO2 EMISSIONS (배출량)
   
## ⚙️ 분석 과정
1. 데이터 구조 파악 및 변수 정의
2. 결측치 및 이상치 처리, 데이터 정제
3. 파생 변수 생성 및 단위 변환
4. EDA를 통한 변수 간 관계 분석
5. 결과 해석 및 인사이트 도출
   
## 📈 주요 결과
- 엔진 크기가 작을수록 연비↑, 배출량↓
- 실린더 수가 적을수록 친환경적
- 변속기 종류에 따라 성능 차이 존재
- 최근 기준 가솔린이 디젤 대비 더 효율적인 경향 확인

## 💡 인사이트 및 제안
- 친환경 차량 설계 조건:
  👉 가솔린 + 3L 이하 엔진 + 5기통 이하 + 무단변속기 + 스몰왜건
- 왜건 차량은 연비와 배출량 측면에서 가장 효율적인 차종으로 나타남
- 국내 시장에서는 인식 부족으로 성장 가능성이 높은 세그먼트로 판단
